# Vegblog API
### 介绍
Self blog API application written with springboot(back-end).
使用 springboot 搭建的个人博客 API 系统（后端）。

### 写该项目的目的
一直以来都是用 .NET 作为开发平台，但是还年轻，也想学习一下生态比较庞大的 Java 平台，都说实战才是最好的学习方式，所以就从一个简单的博客系统入手。

### 已完成的功能
* 用户登录、注册
* 文章的发布、编辑、删除
* 发布文章时标签的添加、删除
* 展示某个标签下的文章
* 跨域支持
* 删除文章时，同时删除文章-标签关联关系
